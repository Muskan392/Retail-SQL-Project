# 🗄️ Retail Sales Analysis (SQL)

## 📌 Overview

This project is a simple SQL-based retail sales analysis.
It uses basic tables like customers, products, and orders to understand sales data and perform some analysis.

---

## 🎯 What I Did

* Created a database and tables
* Inserted sample data
* Used JOINs to combine data
* Calculated total revenue
* Found top customer
* Analyzed product sales

---

## 🛠️ Concepts Used

* CREATE DATABASE / TABLE
* INSERT INTO
* SELECT queries
* INNER JOIN
* GROUP BY
* ORDER BY
* Aggregate functions (SUM)

---

## 🏗️ Tables Used

* Customers
* Products
* Orders

---

## 🚀 How to Run

1. Open MySQL Workbench
2. Open the SQL file
3. Run all queries
